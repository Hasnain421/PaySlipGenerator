# PaySlipGenerator

**PaySlipGenerator** is a simple and efficient Python program designed to calculate and display a complete employee salary slip.  
It includes commission, tax, bonus, allowances, and deductions â€” all computed automatically based on user inputs.

---

##  Features

- Calculates **Commission** based on sales.
- Applies **Tax Slabs** according to total income.
- Adds **Medical** and **Housing Allowances**.
- Deducts **Loan Installments** automatically.
- Awards **Bonus** based on sales performance.
- Displays a clear, formatted **Salary Slip**.

---

##  Formula Highlights

| Component | Description |
|------------|--------------|
| **Commission** | Based on number of units sold |
| **Tax** | Calculated as per total income slab |
| **Allowances** | Medical (15%) and Housing (15%) |
| **Loan Deduction** | 20% of basic salary |
| **Bonus** | Awarded on total sales amount |
